# algorithms study project

## Brief Description
Algorithms study project.

## Compilation
### Requirements
* Compiler support *C++17*.
* [GTest](https://github.com/google/googletest).

### Build
~~~
git clone https://github.com/mbiggio/algorithms.git && \
cd algorithms && \
mkdir build && \
cd build && \
cmake -DCMAKE_BUILD_TYPE=Debug .. && \
make && \
./algorithms_test
~~~
